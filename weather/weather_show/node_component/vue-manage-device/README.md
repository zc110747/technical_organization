基于 Vue3 + pinia + Element Plus 的设备远程管理系统。

## 安装步骤
> 因为使用vite3，node版本需要 14.18+

```bash
#解压，在根目录下安装系统执行需要的环境    
npm install  

#运行
npm run dev

#执行构建命令，生成的dist文件夹放在服务器下即可访问
npm run build
```
